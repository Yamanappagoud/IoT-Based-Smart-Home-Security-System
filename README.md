# IoT-Based-Smart-Home-Security-System
This project is a complete IoT-based smart home security system that integrates hardware, software, and AI to provide real-time surveillance and alerts
The system uses an ESP32-CAM module with a PIR motion sensor to detect movement. When motion is detected, the device captures an image and sends it to a backend server. The server processes the image using the YOLO object detection algorithm and sends an alert via a Telegram bot if any suspicious object (like a person or vehicle) is detected.
